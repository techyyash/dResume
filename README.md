# dResume

Hi there.
This is a Django app I wrote that helps to digitize my resume.

1. Developed a web application enabling users to create personalized resumes with ease.
2. Implemented blog functionality to allow users to share insights and experiences.
3. Enabled resume download feature for users to save and share their resumes conveniently.
4. Functioned as a dynamic portfolio website, showcasing users' professional profiles and accomplishments.
